1.前端安全问题

xss(cross site scripting) /跨站脚本攻击

外部javascript 代码注入到用户界面中

预防措施：
1.过滤转义输入输出
2.避免使用 eval、new Function等执行字符串的方法，除非确定和用户输入无关
3.使用cookie 的 httpOnly属性， js是无法读写的
4.使用innerHTML、document.write 方法的时候 ，对用户输入的字符串 进行转义

CSRF(cross-site request forgery) / 跨站请求伪造

当你进入黑客网站，进行的一些操作会被转移到别的网站上

预防措施：
1.检测网站的 http.referer 是否同域名
2.避免 session 的登陆状态长时间存储在客户端
3.对关键性操作采用验证码 或 taken机制
